# OTP Generator (Python)



This project is a simple OTP (One-Time Password) Generator built with Python. It generates random numeric OTPs of different lengths using Python's random module.

## ✨ Features

* Generate random OTPs<br>
* Supports different OTP lengths<br>
* Simple command-line interface<br>
* Beginner-friendly Python project<br>

## 🛠️ Technologies Used

* Python <br>
* Random Module
